## Hi there 👋
This is the official repository of XJTLU Online Editor containing frontend application and backend applications.

### XJTLU Online Editor
Online learning has become a trend since the breakout of Covid-19. However, there is no platform in the learning mall for XJTLUers to learn programming online. To provide students who are eager to learn programming skills (especially for non-computer majors), our team tries to create a new, interactive way of learning -- a web-based online programming learning platform.  Our goal was to give XJTLUers  the ability to learn the skills they’d need to succeed in the 21st century. Programming contains a world of possibilities — all that’s required is the curiosity and drive to learn.

To this end, our system includes functions such as online editing, real-time feedback interaction, and progress tracking. We will provide a full range of courses from elementary to advanced. Students can have their unique accounts to track progress, record their achievements and find suitable progress. By embedding this system in the learning mall, there is no doubt that all XJTLUers can benefit from it.

During initial period, we will combine our website with the learning mall which is widely applied by students and professors of XJTLU as a pilot project. More specially, on the one hand, we will create a dashboard on the LMO, which guides students in all majors to learn professional programming courses with enjoyable and fulfilling ways. On the other hand, professors in all of majors could use our website to enrich their teaching methods. Through these, about 5000 XJTLUers will make full sense of advantages and superiority of our web-based online programming learning platform and play a magnificent publicity effect.

### Frontend and Backend
Frontend of this project is written in AntdPro, a React-based framework. To start the project, use `npm start:no-mock`.
And backend of this project is written in Django, a powerful web framework in Python. To start the project, use `python manage.py runserver`. 

This project is currently deployed on the remote Huawei Cloud server and successfully being containerized. To access the project, open the url: http://114.115.249.201:8002
